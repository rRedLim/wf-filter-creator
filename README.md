
# Warframe Filter Creator
## *Simple script for creating filters for WF trade chat.*

## How to use

- Open input.txt and write stat combinations
- Open filtercreator.py
- Result will be in output.txt

## Input example

    cc cd ms
    cd ms dmg
    ms fr tox

## Stat keywords

    Critical Chance: cc
    Critical Damage: cd
    Electricity: elec
